# LAFANICA---site-zmeura
Site de prezentare pepiniera de zmeura, realizat in HTML, CSS si JavaScript.
LAFANICA – Site de prezentare pentru cultura de zmeura

Acesta este un site static realizat in HTML, CSS si JavaScript, creat pentru a prezenta activitatea unei pepiniere specializate in cultivarea zmeurei.  
Proiectul include informatii despre soiuri, tehnici de plantare, produse si modalitati de contact.


 Descriere general

Website-ul prezinta:

- Informații despre cultura zmeurei
- Soiuri precum Sokolica, Laszka si Delniwa
- Descrierea pepinierei si experienta acumulata de-a lungul anilor
- Produse si servicii oferite
- Sectiune de contact
- Pagina de logare
- Elemente vizuale moderne si design responsive

Scopul proiectului este de a oferi o prezentare profesionala, simpla si usor de parcurs.

Tehnologii folosite

- **HTML** – structura paginilor
- **CSS** – stilizare, layout, animatii
- **JavaScript** – functionalitati de baza
- **Font Awesome** – iconite pentru social media
- **Design responsive** – adaptare pentru telefon, tableta si desktop
